"ฟังนะ... มันก็แค่เงินน่ะ"
อย่าไปยึดติดกับมันให้มากนัก CHRONOS ถูกสร้างขึ้นมาเพื่อเป็นแค่ 'ตัวกลาง' ให้พวกคุณได้ใช้แลกเปลี่ยนปัจจัยพื้นฐาน และเป็นเครื่องมือให้คุณ 'เก็บออม' ความพยายามของตัวเองเอาไว้ใช้ในวันที่เรี่ยวแรงหมดลงก็เท่านั้น
ผมรู้ว่าพวกคุณบางคนเสพติดการขยายตัวเลขในบัญชี ถ้าคุณมั่นใจในระบบนี้จนอยากจะ 'ลงทุนเพื่อเก็งกำไร' ผมก็ไม่ได้ห้ามหรอกนะเชิญตามสบาย... แต่มันมีกฎเหล็กข้อหนึ่งที่ต่อให้คุณรวยล้นฟ้าแค่ไหนก็หนีไม่พ้น: 'ตายไปคุณก็เอาอะไรไปไม่ได้'
ในระบบของผม เมื่อลมหายใจคุณหยุดลง ตัวเลขพวกนั้นจะกลายเป็นศูนย์ทันที เพราะฉะนั้น แทนที่จะนั่งจ้องกราฟทั้งวัน ออกไปใช้ 'เวลา' ที่เหลืออยู่ทำอะไรที่มีความหมายจริงๆ ซะเถอะ เพราะนั่นคือสมบัติชิ้นเดียวที่คุณครอบครองได้จริงในจักรวาลนี้
— มิสซูบิชิ โตชิบ้า (Mitsubishi Toshiba)
